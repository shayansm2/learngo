## most common formats to use in `Printf`
| symbol   | meaning                                |
|----------|----------------------------------------|
| `%d`     | int base 10                            |
| `%s`     | string                                 |
| `%q`     | double-quoted string                   |
| `%t`     | bool                                   |
| `%.{n}f` | with n decimal points                  |
| `%v`     | **value in default format (any type)** |
| `%T`     | type of the variable                   |
